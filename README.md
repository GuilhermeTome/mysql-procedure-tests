## Getting started

* In first you need to configure a MySQL development zone to use this resources

### Prerequisites

* npm
* gulp client

### Installing

Inside your apache folder. Run these commands

* Clone repository:
```
git clone https://github.com/GuilhermeTome/mysql-procedure-tests.git
```
* Change directory to application folder
```
cd mysql-procedure-tests
```
